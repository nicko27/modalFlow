ModalFlow - Exemples

Cette page présente différents exemples d'utilisation de la bibliothèque ModalFlow. Cliquez sur les boutons pour voir les différentes fonctionnalités en action.

Modal Simple

Une modal basique avec titre et contenu. 
modalFlow.createAndOpen({
    title: 'Modal Simple',
    content: 'Voici une modal simple avec un titre et du contenu.'
});

Modal avec Thème Sombre

Une modal utilisant le thème sombre intégré. 
modalFlow.createAndOpen({
    title: 'Thème Sombre',
    content: 'Cette modal utilise le thème sombre intégré.',
    theme: 'dark'
});

Modal avec Boutons

Une modal avec des boutons personnalisés dans le pied de page. 
modalFlow.createAndOpen({
    title: 'Confirmation',
    content: 'Voulez-vous continuer cette opération ?',
    buttons: [
        {
            text: 'Annuler',
            class: 'modal-flow-button-secondary',
            onClick: () => {
                console.log('Action annulée');
                return true; // Ferme la modal
            }
        },
        {
            text: 'Continuer',
            class: 'modal-flow-button-primary',
            onClick: () => {
                console.log('Action confirmée');
                return true; // Ferme la modal
            }
        }
    ]
});

Une modal que l'on peut redimensionner avec les poignées sur les coins.

Modal avec Formulaire

Une modal contenant un formulaire avec validation. 
modalFlow.createAndOpen({
    title: 'Formulaire',
    content: `
        <form id="exampleForm">
            <div class="form-group">
                <label for="name">Nom</label>
                <input type="text" id="name" class="form-control" required>
            </div>
            <div class="form-group">
                <label for="email">Email</label>
                <input type="email" id="email" class="form-control" required>
            </div>
        </form>
    `,
    buttons: [
        {
            text: 'Annuler',
            class: 'modal-flow-button-secondary'
        },
        {
            text: 'Envoyer',
            class: 'modal-flow-button-primary',
            onClick: () => {
                const form = document.getElementById('exampleForm');
                if (form.checkValidity()) {
                    // Traitement du formulaire
                    return true; // Ferme la modal
                } else {
                    form.reportValidity();
                    return false; // Garde la modal ouverte
                }
            }
        }
    ]
});

Modal avec Onglets

Une modal contenant plusieurs onglets pour organiser le contenu. 
modalFlow.createAndOpen({
    title: 'Modal avec Onglets',
    width: '600px',
    content: `
        <div class="tabs">
            <button class="tab active" data-tab="tab1">Onglet 1</button>
            <button class="tab" data-tab="tab2">Onglet 2</button>
            <button class="tab" data-tab="tab3">Onglet 3</button>
        </div>
        <div class="tab-content active" id="tab1">
            <p>Contenu de l'onglet 1</p>
            <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
        </div>
        <div class="tab-content" id="tab2">
            <p>Contenu de l'onglet 2</p>
            <p>Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
        </div>
        <div class="tab-content" id="tab3">
            <p>Contenu de l'onglet 3</p>
            <p>Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris.</p>
        </div>
    `,
    onOpen: () => {
        // Gestion des onglets
        document.querySelectorAll('.tab').forEach(tab => {
            tab.addEventListener('click', () => {
                // Désactiver tous les onglets
                document.querySelectorAll('.tab').forEach(t => t.classList.remove('active'));
                document.querySelectorAll('.tab-content').forEach(c => c.classList.remove('active'));
                
                // Activer l'onglet cliqué
                tab.classList.add('active');
                document.getElementById(tab.dataset.tab).classList.add('active');
            });
        });
    }
});

Modal avec Contenu Dynamique

Une modal dont le contenu est chargé dynamiquement après ouverture. 
const modal = modalFlow.create({
    title: 'Chargement...',
    content: '<div class="loading">Chargement des données...</div>',
    width: '500px'
});

modal.open();

// Simuler un chargement asynchrone
setTimeout(() => {
    modal.setTitle('Données Chargées');
    modal.setContent(`
        <div>
            <p>Les données ont été chargées avec succès !</p>
            <ul>
                <li>Élément 1</li>
                <li>Élément 2</li>
                <li>Élément 3</li>
            </ul>
        </div>
    `);
}, 2000);

Modal Imbriquée

Une modal qui ouvre une autre modal lorsqu'on clique sur un bouton. 
modalFlow.createAndOpen({
    title: 'Modal Principale',
    content: 'Cliquez sur le bouton ci-dessous pour ouvrir une modal secondaire.',
    buttons: [
        {
            text: 'Ouvrir Modal Secondaire',
            class: 'modal-flow-button-primary',
            onClick: () => {
                modalFlow.createAndOpen({
                    title: 'Modal Secondaire',
                    content: 'Voici une modal secondaire ouverte depuis la première modal.',
                    buttons: [
                        {
                            text: 'Fermer',
                            class: 'modal-flow-button-primary'
                        }
                    ]
                });
                return false; // Ne pas fermer la modal principale
            }
        },
        {
            text: 'Fermer',
            class: 'modal-flow-button-secondary'
        }
    ]
});

modalFlow.createAndOpen({

            <p>Idéale pour les présentations, les galeries d'images ou les interfaces complexes.</p>
        </div>
    `,
    fullscreen: true
});
